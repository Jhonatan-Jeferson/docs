## Request

### Headers

<ParamField header="Authorization" type="string" placeholder="API KEY" required>
  The API key for your account. You can find this in your [account settings](https://squarecloud.app/account/security).
</ParamField>

### Path

<ParamField path="databaseId" type="string" placeholder="Database ID" required>
  The ID of the database.
</ParamField>

## Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if not.
</ResponseField>
<ResponseField name="response" type="array">

The contents of the response.
<Expandable title="Toggle object">
<ResponseField name="date" type="string">
  The date in ISO format.
</ResponseField> 
<ResponseField name="cpu" type="string">
  The current CPU usage percentage. 
</ResponseField>
<ResponseField name="ram" type="string">
  The current RAM usage 
</ResponseField>
<ResponseField name="net" type="int[]">
  An array representing total network used. First the input, second the output. 
</ResponseField>
</Expandable>
</ResponseField>
