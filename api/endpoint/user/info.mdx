---
title: "Get User"
api: "GET https://api.squarecloud.app/v2/user"
authMethod: "authorization"
description: "Get information about your account or another user."
---

### Query Parameters
**/v2/user/:userID**

This is the ID of the user you want to get information about. If you want to get information about your own account, you can leave this blank.
